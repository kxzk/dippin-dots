#### github cli
```bash
gh auth login
gh config set editor nvim
gh config set pager delta
```

#### atuin
```bash
curl --proto '=https' --tlsv1.2 -LsSf https://setup.atuin.sh | sh
atuin register -u <USERNAME> -e <EMAIL>
atuin import auto
atuin sync
```
